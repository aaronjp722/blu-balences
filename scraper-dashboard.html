Schedule Configuration

Saved to your Supabase scraper_config table. Restart your Railway service after saving.

6 = every 6 hrs · 24 = daily · 168 = weekly

To apply: Railway → -yp-scraper-agent → Deployments → Restart. The agent reads the config from Supabase on startup.

Add this to scraper_agent.py to read schedule from Supabase

def load_config():
    try:
        resp = supabase.table("scraper_config").select("*").execute()
        return {r["key"]: r["value"] for r in resp.data}
    except Exception as e:
        log.error(f"Could not load config: {e}")
        return {}

cfg            = load_config()
SCHEDULE       = cfg.get("SCHEDULE", os.environ.get("SCHEDULE", "weekly"))
RUN_HOUR       = cfg.get("RUN_HOUR",  os.environ.get("RUN_HOUR",  "06:00"))
INTERVAL_HOURS = int(cfg.get("INTERVAL_HOURS", os.environ.get("INTERVAL_HOURS", "24")))
